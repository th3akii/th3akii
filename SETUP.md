# GitHub profile README setup

1. Napravi public repo koji se zove isto kao tvoj GitHub username.
   Primer: ako ti je username `TVOJ_USERNAME`, repo treba da bude `TVOJ_USERNAME/TVOJ_USERNAME`.

2. U root repo-a ubaci ova 3 fajla:
   - `README.md`
   - `dark_mode.svg`
   - `light_mode.svg`

3. U sva 3 fajla zameni `TVOJ_USERNAME` svojim stvarnim GitHub username-om.

4. Commituj i pushuj na `main`.

5. Otvori svoj GitHub profil i proveri kako izgleda u dark i light modu.

Šta je promenjeno:
- izbačen je `Contact` blok
- ostavljen je samo projekat `NextWatch`
- tekst je više prilagođen studentu i backend učenju
- promenjena je color schema da ne bude identična referenci

Kasnije, ako hoćeš, možemo dodati i:
- automatske GitHub stats
- dnevni update preko GitHub Actions
- još bolji light mode
- preciznije podatke za NextWatch
